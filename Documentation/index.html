How to change AdMob publisher id?

  1. Open Eclipse > StickHero> src > main.lua
  2. Enter your AdMob publisher id inside ENTER_YOUR_ANDROID_ADMOB_APP_ID_HERE

How to setup Google Play leaderboard

  1. Open Eclipse > StickHero> src > main.lua
  2. Open Eclipse > StickHero> src > build.settings
  3. ENTER_YOUR_ANDROID_LEADERBOARD_ID_HERE (main.lua)
  4. ENTER_YOUR_GAMES_APP_ID_HERE (build.settings)

How to change brick color?

  1. Open Eclipse > StickHero> src > scenes/game.lua
  2. Enter your color inside
  3. player:setFillColor(38, 38, 38)
  4. Note:The color must be RGB

How to change line color?

  1. Open Eclipse > StickHero> src > scenes/game.lua
  2. Go to row 421
  3. Enter your color inside
  4. planks[#planks]:setFillColor ( 22, 160, 133)
  5. Note:The color must be RGB
